# 🛒 E-Commerce Sales Performance Analytics | Power BI

---

## 🚀 Project Overview

This project presents an **end-to-end Business Intelligence solution** developed using **Power BI** to analyze and visualize E-Commerce sales performance data.

The objective of this project is to transform raw transactional sales data into meaningful business insights using:

- Data Cleaning
- ETL (Extract, Transform, Load)
- Dimensional Data Modeling
- KPI Analysis
- Interactive Dashboarding

This solution enables data-driven decision-making by identifying key trends in sales, profit, customer behavior, and payment preferences.

---

## 🎯 Problem Statement

E-Commerce platforms generate large volumes of transactional data that often lack analytical visibility without structured modeling.

The goal of this project is to:

- Analyze overall sales performance
- Track monthly profit trends
- Identify high-performing product categories
- Understand customer purchasing patterns
- Evaluate state-wise revenue contribution
- Analyze payment mode distribution
- Detect profit-driving sub-categories

## 🏗️ Data Modeling Approach

A **Dimensional Data Model (Star Schema)** was implemented for efficient analytical querying.

### Fact Table:
- Sales Transaction Data

### Dimension Tables:
- Dim_Customer
- Dim_Product
- Dim_State
- Dim_PaymentMode
- Dim_Date

This approach ensures:

- Optimized Query Performance  
- Scalable Reporting  
- Data Consistency  
- Faster Aggregation
